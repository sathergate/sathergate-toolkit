ratelimit-next MCP

Rate limiting for Next.js

Three algorithms. Zero dependencies. Every endpoint protected.

$ npm install ratelimit-next

Quick start

1
Define rules
lib/rate-limit.ts 
import { createFloodgate } from "ratelimit-next";

export const gate = createFloodgate({
  rules: {
    api: { limit: 100, window: "1m" },
    auth: { limit: 5, window: "15m" },
  },
});
2
Protect routes
app/api/data/route.ts 
import { withRateLimit } from "ratelimit-next/next";
import { gate } from "@/lib/rate-limit";

export const GET = withRateLimit(gate, "api", async (req) => {
  return Response.json({ data: "hello" });
});
3
Standard headers, automatic
// Every response includes:
// X-RateLimit-Limit: 100
// X-RateLimit-Remaining: 99
// X-RateLimit-Reset: 1711468800
// 429 + Retry-After when exceeded
