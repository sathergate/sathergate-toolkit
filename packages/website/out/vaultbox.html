vaultbox

MCP

Encrypted secrets for Next.js. Store encrypted secrets in your repo. Decrypt at runtime. Zero external dependencies.

$npm install vaultbox

Quick start

1

Initialize

terminal
npx lockbox init
npx lockbox set DATABASE_URL "postgresql://user:pass@host/db"
npx lockbox set API_KEY "sk-secret" --env production
2

Use in code

lib/secrets.ts
import { createLockbox } from "vaultbox";

const box = createLockbox();
const dbUrl = box.require("DATABASE_URL");
const apiKey = box.secret("API_KEY");
3

Commit safely

.gitignore
# .secrets/*.json is safe to commit (encrypted)
# .lockbox-key is NEVER committed
# Share the key via 1Password, then:
LOCKBOX_KEY=your-hex-key npm run build
