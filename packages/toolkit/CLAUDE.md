## What This Is
@sathergate/toolkit - Meta-package and unified MCP server for the sathergate-toolkit. Includes gatehouse for drop-in RBAC with role hierarchy. See the website package for documentation site.
## MCP Tools
- list_packages - List all 8 packages
- get_package - Detailed info on one package
- find_package - Describe a need, get a recommendation
- quick_start - Copy-paste-ready code
